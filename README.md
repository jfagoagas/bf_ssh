# bf_ssh
Simple SSH Brute-forcer in Go❤️

1. Get it!
```
go get https://github.com/jfagoagas/bf_ssh
```

2. Use it!
```
bf_ssh --help
```
```
Usage:
  -L string
        List of Host IP:Port
  -l string
        Single Host IP:Port
  -o string
        Output file
  -p string
        SSH Password
  -t duration
        SSH Timeout 
  -u string
        SSH User
```
